# 📚 Book Helper

**Book Helper** — это Telegram-бот для подбора книжных рекомендаций на основе пользовательских предпочтений.

Пользователь указывает любимых авторов и жанры, а бот формирует персональные рекомендации с помощью YandexGPT.

---

## ✨ Возможности

- регистрация пользователя при первом запуске бота;
- сохранение любимых авторов;
- сохранение любимых жанров;
- генерация персональных книжных рекомендаций;
- интеграция с YandexGPT;
- хранение пользовательских данных в PostgreSQL;
- ограничение частоты запросов для защиты от спама;
- простая архитектура с разделением на слои.

## ⚙️ Конфигурация

Для запуска проекта необходимо создать конфигурационный файл:

```
configs/book-helper.toml
```

Файл должен содержать параметры подключения к базе данных, токен Telegram-бота и данные для обращения к YandexGPT.

Пример содержимого `book-helper.toml`:

```toml
bind_addr = ":8080"
db_url = "postgres://user:password@localhost:5432/book_helper?sslmode=disable"
tg_token = "YOUR_TELEGRAM_BOT_TOKEN"

[rec_data.gpt_data]
gpt_token = "YOUR_YANDEX_GPT_TOKEN"
model = "yandexgpt-lite"
temperature = 0.7
max_tokens = 1000
prompt = "Подбери пользователю книги на основе его любимых авторов и жанров."
```

Где:

| Поле          | Описание                                       |
| ------------- | ---------------------------------------------- |
| `bind_addr`   | адрес, на котором может запускаться приложение |
| `db_url`      | строка подключения к PostgreSQL                |
| `tg_token`    | токен Telegram-бота                            |
| `gpt_token`   | токен для доступа к YandexGPT                  |
| `model`       | используемая модель YandexGPT                  |
| `temperature` | параметр разнообразия ответа модели            |
| `max_tokens`  | максимальное количество токенов в ответе       |
| `prompt`      | системный промпт для генерации рекомендаций    |

Проект является учебным и демонстрирует реализацию Telegram-бота на Go с использованием PostgreSQL и внешнего AI-сервиса для генерации книжных рекомендаций.

Дальнейшее развитие проекта не планируется в связи с ограничениями и нестабильностью работы Telegram в РФ.
